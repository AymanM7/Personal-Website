# Ayman Portfolio 

This project is a modern developer portfolio website built with Next.js and Tailwind CSS. It is designed to showcase my skills, experience, projects, and blogs in a visually appealing and responsive manner.

## Features
- **Next.js**: Utilizes the powerful React framework for server-side rendering and static site generation.
- **Tailwind CSS**: Provides utility-first CSS for rapid UI development and customization.
- **Responsive Design**: Works seamlessly across devices of all sizes.
- **Animated Components**: Engaging UI elements for a dynamic user experience.
- **Sections Included**:
  - About
  - Experience
  - Education
  - Skills
  - Projects
  - Blogs
  - Contact

## Folder Structure
- `public/`: Static assets (images, icons, text prompts)
- `src/components/`: Reusable React components for different sections
- `src/pages/`: Next.js pages (about, blogs, contact, projects, etc.)
- `src/styles/`: Global and module CSS files

## Getting Started
1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the development server**:
   ```bash
   npm run dev
   ```
3. **Open [http://localhost:3000](http://localhost:3000)** to view the site.

## Customization
- Update content in the `public/All-Texts/` folder for texts and prompts.
- Add your images to the `public/images/` directory.
- Modify components in `src/components/` to personalize sections.
